## 建议拆分清单（可直接复制）

### FE-01 目录与出口工单
- 目标：创建 admin/avatar 独立子模块目录与出口。
- 验收：avatar 逻辑不散落在 admin 主文件。

### FE-02 类型与服务工单
- 目标：扩展 Admin 头像字段并新增 avatar 上传服务。
- 验收：list/detail 映射含 avatar_small_url、avatar_large_url、avatar_version、avatar_updated_at。

### FE-03 弹窗裁剪工单
- 目标：实现 1:1 拖拽裁剪和 32/128 双预览。
- 验收：预览与最终落盘一致。

### FE-04 编辑页接入工单
- 目标：Admin 编辑页增加“编辑头像”按钮并接入弹窗。
- 验收：上传成功后编辑态与列表态均刷新。

### FE-05 异常处理工单
- 目标：覆盖格式/大小/越界/权限/网络错误提示。
- 验收：失败不关闭弹窗，支持直接重试。

### BE-01 PostgreSQL migration 工单
- 目标：新增 admin 头像相关字段并提供回滚。
- 验收：迁移可重复执行且可回滚。

### BE-02 上传接口工单
- 目标：实现 POST /admin/{id}/avatar 与参数校验。
- 验收：非法请求返回明确业务错误码。

### BE-03 图像处理工单
- 目标：输入 bmp/jpg/png，输出 jpg 32x32 + 128x128。
- 验收：尺寸精确，质量参数固定。

### BE-04 存储与静态路由工单
- 目标：写入 uploads/avatar/{admin_id}/ 并挂载 /static/avatar。
- 验收：URL 可访问，服务账号可写。

### BE-05 列表与详情扩展工单
- 目标：admin/list 与 admin/{id} 回包新增头像字段。
- 验收：前端无需额外兼容旧语义。

### BE-06 新建管理员默认头像工单
- 目标：新建管理员未上传头像时自动分配灰色人头默认头像（32/128）。
- 验收：新建后列表与编辑态均不出现空头像。

### INT-01 正向链路联调工单
- 目标：编辑页上传裁剪保存全流程打通。
- 验收：列表显示 small，编辑态显示 large。

### INT-02 异常链路联调工单
- 目标：验证非法格式、超大小、越界、越权、admin 不存在。
- 验收：错误码与提示文案一致。

### INT-03 一致性联调工单
- 目标：数据库、接口、文件系统三方一致。
- 验收：抽样检查通过。

### INT-04 缓存联调工单
- 目标：版本更新后不读旧头像。
- 验收：连续替换头像后页面均展示最新图。

### INT-05 默认头像联调工单
- 目标：验证默认头像回退逻辑。
- 验收：新建管理员不上传头像显示灰色默认头像；删除自定义头像后回退默认头像。
